ABOUT PROJECT

The Intelligent Metro Flow Optimization Platform is a web application that analyzes metro passenger travel patterns. The system allows users to enter journey information such as start station, stop station, trip type, travel time, and passenger count. Based on this data, the platform generates passenger summaries and traffic analysis to help visualize metro usage. This project demonstrates the use of modern web technologies including HTML, CSS, and JavaScript to build smart transportation applications.
